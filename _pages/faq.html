FAQ

  • How do I use Hyde?
  • Though there isn't an installer yet, using Hyde is simple. To use Hyde on any platform, you must first install npm and download Hyde from the project repo. Then simply run the following command from within the project directory: 
    $ npm install && npm start
  • What is the keyboard shortcut to (...)?
  • All of Hyde's keyboard shortcuts can be found here.
  • How do I set a YAML-frontmatter template for Jekyll blog posts?
  • To set a frontmatter template for Jekyll posts, create a .yml file with only the frontmatter used in each of your Jekyll blog posts. If you'd prefer you can also use the default template provided here. Once this file has been created you can set this as your frontmatter template by opening Hyde's settings menu (ctrl + ,) and pasting the path to your created file into the textbox labeled Frontmatter Template.
